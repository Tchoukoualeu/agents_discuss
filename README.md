# DebateForge

Multi-agent debates with OpenAI streaming and MongoDB storage. Next.js app.

## Run

```bash
npm install
cp .env.example .env.local   # fill in values
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Env

`MONGODB_URI`, `MONGODB_DB` (optional), `OPENAI_API_KEY`, `OPENAI_MODEL` (optional). See `.env.example`.

## Build

```bash
npm run build
npm run start
```
